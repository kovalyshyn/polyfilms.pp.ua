# PolyFilm Ortho 🎞️

**PolyFilm Ortho** — це проект відновлення та розробки крафтової ортохроматичної фотоемульсії для середнього формату (120 тип).

Цей проект є триб’ютом фотографічним технологіям кінця XIX – початку XX століття, відтвореним за допомогою сучасних безпечних компонентів без використання токсичних речовин та з використанням срібла, відновленого з відпрацьованого фіксажу.

## 🧪 Технічні характеристики

* **Назва:** PolyFilm Ortho (Experimental Batch)
* **Тип:** Ортохроматична чорно-біла плівка
* **Номінальна чутливість:** ISO 20 / 14° DIN
* **Формат:** 120 (Medium Format)
* **Особливість:** Чутлива до синього та зеленого спектрів; нечутлива до червоного (darkroom safe).

## 🛠️ Рекомендації з обробки

Для досягнення найкращих результатів рекомендується підтримувати температуру розчинів **18-20°C**.

| Проявник         | Розведення        | Час (18°C) |
| :---             | :---              | :---       |
| **Rodinal**      | 1:25              | 13 - 18 хв |
| **Kodak HC-110** | Dilution B (1+31) | 12 – 16 хв |

> **Важливо:** При штучному "теплому" освітленні (лампи розжарювання) чутливість падає у 2–3 рази. Не використовуйте червоні або помаранчеві фільтри.
